# Saddle

`Saddle` can be added to creatures that do not have one already using a custom MonsterDB.Saddle script. Simply add the field `Saddle` to any creature and modify the fields.

```yml
hoverText: string
maxUseRange: float
detachOffset: Vector3
attachAnimation: string
maxStamina: float
runStaminaDrain: float
swimStaminaDrain: float
staminaRegen: float
staminaRegenHungry: float
drownEffects: EffectList
mountIcon: string             # Sprite ID
```

See file `Resources.Sprites.Names.txt` for available sprite IDs
